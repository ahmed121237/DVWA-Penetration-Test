# DVWA Penetration Test

## Overview
This project documents a structured penetration test conducted against Damn Vulnerable Web Application (DVWA).

## Scope
- SQL Injection
- Reflected XSS
- Stored XSS
- IDOR / File Inclusion
- Broken Authentication
- Command Injection

## Methodology
- Reconnaissance
- Enumeration
- Exploitation
- Risk Assessment
- Reporting

## Tools Used
- Burp Suite
- SQLMap
- Nikto
- Gobuster
- Wireshark

## Deliverables
- Professional Penetration Testing Report
- Risk Assessment
- Remediation Recommendations

## Disclaimer
Testing was conducted in a controlled lab environment against DVWA for educational purposes.
